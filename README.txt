<h2>Java Portion</h2> 
<p>The Java portion consists of 3 packages within the main folder, and 2 packages within the test folder.<br>
In order to run the tests use the command: <strong>mvn install</strong> within the JavaAST folder.<br>
In order to run the program, navigate to the target folder and run the JavaEval.jar using the command:  
<strong>java -jar JavaEval.jar directoryName</strong></p>
<h2>Python Portion</h2>
<p>The Python portion consists of 2 packages with source code, a main.py and a test.py.<br>
Before the program is run, the requirements.txt must be executed using the command: <strong>pip install -r requirements.txt</strong><br>
In order to run the main.py simply run python main.py within the src folder.<br>
In order to run the tests simply run python tests.py within the src folder</p>
